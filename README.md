# Hoponai v2 — hoponai.com

Enterprise AI training platform. Interactive learning that adapts to every employee.

## Pages

```
/                        → Homepage
/solutions/onboarding    → Employee Onboarding
/solutions/training      → Continuous Training
/solutions/customers     → Customer Training
/solutions/sops          → SOPs & Compliance
```

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub: `gh repo create hoponai-v2 --public --push`
2. Import at vercel.com/new
3. Add hoponai.com domain in Vercel Settings → Domains
